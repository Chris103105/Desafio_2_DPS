##  Características
- **Gestión de Piezas:** Registro de nombre, marca, serie, precio y fecha.

- **Interactividad:** Uso de `FlatList`, `Modal`, `Picker` y `DateTimePicker`.

##  Instalación y Configuración

Sigue estos pasos para clonar el proyecto y ejecutarlo en tu entorno local.

### 1. Requisitos Previos
Asegúrate de tener instalado:
- [Node.js](https://nodejs.org/) (Versión LTS recomendada)
- [Git](https://git-scm.com/)
- La app **Expo Go** en tu celular (disponible en App Store y Play Store) o un emulador configurado.

### 2. Clonar el Repositorio
Abre tu terminal y ejecuta los siguientes comandos:

```bash
# Clonar el proyecto
git clone https://github.com/Chris103105/Desafio_2_DPS.git

# Entrar a la carpeta del proyecto
cd Desafio2_DPS

# Instalar dependencias base
npm install

# Instalar librerías específicas si no se instalaron automáticamente
npx expo install @react-native-picker/picker @react-native-community/datetimepicker
# Ejecucion 
npx expo start

Para probarlo:

Escanea el código QR que aparece en la terminal usando la app Expo Go (Android) o la cámara (iOS).

Asegúrate de que tu celular y tu computadora estén conectados a la misma red Wi-Fi.
